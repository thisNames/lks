这位更是重量级的（w(ﾟДﾟ)w）

为目录里指定拓展名的文件，创建符号链接。这样可以大大的节约磁盘的存储空间
可开启递归模式，这样收集的文件更多
设置递归深度，藏在非常的！里面的！角落里的文件也能找出来哦
设置文件最大收集数量，可限制文件太多

基本使用：*
指定拓展名：* .vpk
开启递归：* .vpk -R
开启递归并输出操作日志：* .vpk -R -L
开启递归，并输出操作日志，但是不执行符号链接操作：* .vpk -R -L -SC

.vpk 指定拓展名（默认就是 .vpk）
-R -L -SC 开关指令
这里的 * 即是包罗万象（也就是源目录的路径罢了(^A^)） 

支持的设置：[保存日志] [启用递归] [仅显示收集]

## 命令

### `lks`
`lks`命令用于批量创建符号链接。

#### 用法
```bash
lks [源目录] [目标目录]
```

#### 参数
- `[源目录]`: 需要创建符号链接的源目录。
- `[目标目录]`: 符号链接的目标目录。

### `lk`
`lk`命令用于创建单个符号链接。

#### 用法
```bash
lk [源文件] [目标文件]
```

#### 参数
- `[源文件]`: 需要创建符号链接的源文件。
- `[目标文件]`: 符号链接的目标文件。

## 备注
- 该工具仅支持Windows系统。
- 创建符号链接需要管理员权限。
